# Contact Center AI

AI-native Contact Center as a Service (CCaaS) platform.

---

## Overview

Contact Center AI provides a complete platform for customer service operations:

- [Agent Console](/ai-for-service/contact-center/agent-console) — Unified agent desktop
- [Routing](/ai-for-service/contact-center/routing) — Intelligent skill-based routing
- [Queue Management](/ai-for-service/contact-center/queues) — Virtual waiting areas
- [Skills](/ai-for-service/contact-center/skills) — Agent capability management
- [Supervisor Tools](/ai-for-service/contact-center/supervisor) — Real-time monitoring
- [Campaigns](/ai-for-service/contact-center/campaigns) — Outbound voice and SMS
- [Case Management](/ai-for-service/contact-center/case-management) — Ticket lifecycle

## Quick Start

### 1. Configure Queues

Set up queues for different inquiry types:

```yaml
Queue: Customer Support
Hours: Mon-Fri 8am-8pm EST
Routing: skill-based
Max wait: 5 minutes
```

[Learn more about Queue Management →](/ai-for-service/contact-center/queues)

### 2. Define Skills

Create skills to categorize agent capabilities:

```yaml
Skill Group: Products
Skills:
  - Enterprise Software (Novice → Expert)
  - Hardware Support (Novice → Expert)
```

[Learn more about Skills →](/ai-for-service/contact-center/skills)

### 3. Configure Routing

Set up routing rules to match customers with the right agents:

```yaml
Routing Rule: Technical Issues
Required skills: Technical Support (Good+)
Language: Match customer
Priority: Based on customer tier
```

[Learn more about Routing →](/ai-for-service/contact-center/routing)
