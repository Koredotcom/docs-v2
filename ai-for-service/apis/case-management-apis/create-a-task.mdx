Creates a new task within the case management system.

| Field | Value |
|---|---|
| **Method** | POST |
| **Endpoint** | `https://{{host}}/caseManagement/api/public/analytics/{{streamId}}/createTask` |
| **Content Type** | `application/json` |
| **Authorization** | `auth: {{JWT}}` See [How to generate the JWT Token](/ai-for-service/apis/automation/api-introduction#generating-the-jwt-token). |
| **API Scope** | [Case Management Configuration](/ai-for-service/apis/automation/api-introduction#associating-api-scopes) |

## Path Parameters

| Parameter | Required | Description |
|---|---|---|
| `host` | Required | Environment URL. For example, `https://platform.kore.ai`. |
| `streamId` | Required | Bot identifier or Stream identifier. Access it from the General Settings page of the bot. For example, `st-084d3c5e-376a-559f-9987-a012bb2bxxxx`. |

## Header Parameters

| Parameter | Required | Description |
|---|---|---|
| `accountId` | Optional | The account identifier associated with the API request. For example, `6639f7bd58c97ffac03bxxxx`. |

## Request Body Parameters

| Parameter | Required | Type | Description |
|---|---|---|---|
| `caseId` | Required | String | Unique case identifier. |
| `caseRefId` | Required | String | Case reference identifier. |
| `taskTmp` | Required | String | Unique task template identifier. |
| `name` | Required | String | Task name. |
| `userInfo` | Required | Object | User information object. |
| `userInfo._id` | Optional | String | Unique user identifier. If not provided, you must include either `emailId` or `phoneNumber`. |
| `userInfo.name` | Required | String | User's name. |
| `userInfo.emailId` | Optional | String | User's email address. Mandatory if `_id` is not entered. |
| `userInfo.phoneNumber` | Optional | String | User's phone number. Mandatory if `_id` is not entered. |
| `source` | Required | String | Source of the case. For example, `rtm`. |
| `desc` | Required | String | Task description. |
