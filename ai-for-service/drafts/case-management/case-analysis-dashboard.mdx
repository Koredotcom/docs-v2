The Analyze Cases dashboard provides comprehensive insights into case and task metrics. App owners, supervisors, and agents can filter, monitor, and track metrics based on their role.

## Navigation

Go to **Contact Center AI > Case Management > Analyze > Dashboard**.

![Dashboard](/ai-for-service/drafts/case-management/images/dashboard.png)

The dashboard contains four sections: **Calendar**, **Channels**, **Cases**, and **Tasks**.

---

## Filters

### Calendar

Use the calendar to select a date range for interaction and case metrics. The default range is always the current date.

Available time ranges (based on the agent's time zone):

- Today
- Yesterday
- Last 7, 28, or 30 days (excluding today)
- Custom range: up to 31 days, from 12:00 AM to 11:59 PM

![Calendar](/ai-for-service/drafts/case-management/images/dashboard-calendar.png)

### Channel

Filter agent interaction data by channel type using the channel selector in the top-right corner of the dashboard. Available channels: **Call**, **SMS**, **Voice**, and **Email**.

![Channel Filter](/ai-for-service/drafts/case-management/images/channels.png)

---

## Case Metrics

Supervisors can use this section to filter, review, and monitor individual cases and tasks assigned to agents.

### Cases

The dashboard tracks the following metrics for cases:

![Cases](/ai-for-service/drafts/case-management/images/channels.png)

| Metric | Description |
|---|---|
| **Total** | Real-time count of all cases. |
| **Reopened** | Frequency of cases reopened by agents, indicating potential issues with initial resolutions. |
| **Unassigned** | Number of cases not yet assigned to an agent. |
| **Closed** | Number of cases closed. |
| **Average Resolution Time** | Average time taken to resolve cases from creation. |
| **Cases Created by Day** | Volume of new cases created based on the selected date range. |
| **Status** | Case distribution by status: **Open**, **In Progress**, **On Hold**, and **Resolved**. |
| **Priority** | Case distribution by priority: **Critical**, **High**, **Medium**, and **Low**. |

### Tasks

The dashboard tracks the following metrics for tasks:

![Tasks](/ai-for-service/drafts/case-management/images/dashboard-tasks.png)

| Metric | Description |
|---|---|
| **Total** | Total number of tasks assigned. |
| **Unassigned** | Number of tasks not yet assigned. |
| **Closed** | Number of tasks closed. |
| **Average Resolution Time** | Average time taken to resolve tasks from creation to closure. |
| **Tasks Created by Day** | Volume of new tasks generated based on the selected date range. |
| **Status** | Task distribution by status: **Open**, **In Progress**, **On Hold**, and **Resolved**. |
| **Priority** | Task distribution by priority: **Critical**, **High**, **Medium**, **Low**, and **No Priority**. |
