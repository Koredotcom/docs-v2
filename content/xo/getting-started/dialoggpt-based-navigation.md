# Redesigned Navigation for DialogGPT-based App

The redesigned navigation experiences for DialogGPT-based apps improve discoverability and help users navigate the AI Agent creation process with greater clarity and efficiency.

## Key Navigation Changes

### 1. Features Grouped Under New Sections  

In the redesigned navigation, all features are grouped into the following sections:

* Orchestration
* Evaluation
* Conversation Management 

<img src="../images/key_navigation_dialoggpt.png" alt="key_navigation_dialoggpt" title="key_navigation_dialoggpt" style="border: 1px solid gray;zoom:50%;"/>  
 
### 2. Use Cases Renamed to Orchestration

* Small Talk, Digital Skills, and Alert Tasks are moved under **Conversation Management > Settings**.
* The Feedback Survey has been moved to Conversation Management.

<img src="../images/orchestration.png" alt="orchestration" title="orchestration" style="border: 1px solid gray;zoom:50%;"/>

### 3. Testing Renamed to Evaluation

A new **Playground** section in the **Evaluation** menu combines Debug Logs and Talk to Bot in a side-by-side view, providing an integrated testing and debugging experience with synchronized session control and status management. [Learn more](../automation/testing/playground.md).
