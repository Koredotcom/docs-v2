# Quality AI API List

The following table lists the APIs available for Quality AI.


| API            | Description                                 | App Scope         |
|----------------|---------------------------------------------|-------------------|
| [Raw Data API](./raw-data-api.md) | Provides detailed Quality AI and Conversation Intelligence data, including quality metrics and analytics for each interaction | Configuration |
| [Auto QA Reporting API](auto-qa-reporting-api.md) | Provides interaction-level Auto QA scoring and metric qualification based on the associated Quality AI form | Configuration |
