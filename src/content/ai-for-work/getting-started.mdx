# Getting Started

Set up AI for Work and deploy your first agent.

---

## Prerequisites

- Kore.ai account with AI for Work access
- Admin permissions for initial setup
- (Optional) Microsoft Teams or Slack workspace for deployment

---

## Access AI for Work

1. Navigate to the Kore.ai platform
2. Select **AI for Work** from the product menu
3. Choose or create a workspace

---

## Quick Setup

### Step 1: Create a Workspace

Workspaces organize agents, users, and settings for your organization.

```
Workspace Settings:
├── Name: "Engineering Team"
├── Description: "AI assistants for engineering workflows"
└── Members: Invite team members
```

### Step 2: Choose an Agent Type

| Option | Best For |
|--------|----------|
| **Pre-built Solution** | Quick deployment with IT Assist or HR Assist |
| **Pre-built Agent** | Customizable templates for common use cases |
| **Custom Agent** | Full control over agent behavior and knowledge |

### Step 3: Connect Knowledge

Link your enterprise knowledge sources:

- **Search AI** — Connect Confluence, SharePoint, Google Drive
- **Agentic Apps** — Use Agent Platform apps as knowledge sources
- **Amazon Q** — Connect Amazon Q for AWS-based knowledge

### Step 4: Configure Channels

Deploy to your preferred channels:

1. Navigate to **Assist Configuration** → **Channels**
2. Select channels (Teams, Slack, Web)
3. Follow channel-specific setup instructions
4. Test the deployment

---

## Your First Agent

### Using a Pre-built Solution

1. Go to **Pre-built Solutions**
2. Select **IT Assist** or **HR Assist**
3. Configure the solution settings
4. Connect required integrations
5. Deploy to channels

### Creating a Custom Agent

1. Go to **Custom Agents** → **Create Agent**
2. Select agent type (Prompt, API, Search, etc.)
3. Configure:
   - **Name and description**
   - **Instructions** — Define agent behavior
   - **Knowledge** — Connect data sources
   - **Tools** — Add integrations
4. Test in the preview panel
5. Deploy to channels

---

## AI for Work Assistant

The AI for Work Assistant provides a unified interface for accessing all deployed agents.

### Features

- **Unified access** — Single entry point for all agents
- **Scheduling** — Book meetings and manage calendar
- **Context switching** — Seamlessly move between agents
- **History** — Access previous conversations

### Enable the Assistant

1. Go to **Assist Configuration** → **AI for Work Assistant**
2. Enable the assistant
3. Configure available agents
4. Set up scheduling integrations (optional)

---

## Marketplace

Browse and install pre-built agents and solutions from the Marketplace.

1. Navigate to **Marketplace**
2. Browse categories or search
3. Preview agent capabilities
4. Install to your workspace
5. Configure and deploy
