## Platform Capabilities

The Agent Platform is a unified framework for developing, deploying, and managing AI-powered applications. Build everything from simple chatbots to complex multi-agent systems.

### [Multi-Agent Orchestration](/agent-platform/orchestration)
Build intelligent systems where multiple AI agents collaborate autonomously. Choose supervisor or adaptive network patterns based on your needs.

### [Tools & Integrations](/agent-platform/tools)
Connect agents to external systems with workflow tools, code tools, and MCP protocol. Access any API, database, or enterprise service.

### [Knowledge & RAG](/agent-platform/knowledge)
Enable context-aware responses with retrieval-augmented generation. Connect 100+ data sources including Confluence, SharePoint, and databases.

### [Model Hub](/agent-platform/models)
Use any AI model—OpenAI, Anthropic, Google, or custom fine-tuned models. Manage deployments and optimize for cost and performance.

### [Prompt Studio](/agent-platform/prompts)
Design, test, and optimize prompts collaboratively. Compare across models, use variables, and deploy with version control.

### [Evaluation Studio](/agent-platform/evaluation)
Measure AI performance with built-in and custom evaluators. Test models and full agentic workflows before deployment.

### [AI Safety & Guardrails](/agent-platform/guardrails)
Built-in protection for toxicity, PII, prompt injection, and bias. Configure input/output scanning for compliance.

### [Analytics & Observability](/agent-platform/analytics)
Monitor performance with traces, sessions, and metrics. Debug issues and optimize with comprehensive logging.

## Build Your First Agent

### 1. Create an Agentic App

```yaml
name: Customer Support Bot
description: Handles customer inquiries about orders and products
model: gpt-4o
```

### 2. Add an Agent

```yaml
name: Order Assistant
scope: Order status, tracking, returns, modifications
instructions: |
  You help customers with order-related questions.
  Always verify the order number before providing details.
  Be concise and friendly.
```

### 3. Connect Tools

```yaml
tools:
  - get_order_status    # Fetch order details
  - track_shipment      # Get tracking info
  - initiate_return     # Start return process
```

### 4. Deploy

Publish to web widget, API, Slack, Teams, or voice channels.

[**Start the Quickstart →**](/agent-platform/quickstart)
