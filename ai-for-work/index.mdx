## Platform Capabilities

AI for Work enables organizations to deploy AI-powered assistants that connect to enterprise systems and automate common workflows.

### [Custom Agents](/ai-for-work/custom-agents)
Build and configure AI agents tailored to your organization's needs. Connect enterprise knowledge sources and define agent behavior.

### [Pre-built Agents](/ai-for-work/prebuilt-agents)
Deploy configurable agents designed for quick deployment and workflow integration with minimal setup.

### [Pre-built Solutions](/ai-for-work/solutions)
Domain-specific automation packages including IT Assist for IT support and HR Assist for employee services.

### [Analytics](/ai-for-work/analytics)

Monitor agent performance with dashboard and data views. Track usage, conversations, and outcomes.

## Agent Types

| Agent Type | Description | Use Case |
|------------|-------------|----------|
| **Prompt Agent** | LLM-powered responses with custom instructions | General Q&A, content generation |
| **API Agent** | Connect to external APIs and services | Data retrieval, system integration |
| **Search Agent** | RAG-based retrieval from knowledge sources | Enterprise search, documentation |
| **Agentic Flow Agent** | Multi-step workflows with decision logic | Complex processes, approvals |
| **Autonomous Agent** | Self-directed task completion | Research, analysis, automation |
| **Bot Agent** | Conversational dialog with structured flows | Guided interactions, forms |
| **Workflow Agent** | Visual workflow automation | Business process automation |
| **MCP Agent** | Model Context Protocol integrations | External tool connections |

## Deployment Channels

Deploy AI for Work agents across multiple channels:

- **Microsoft Teams** — Native Teams app integration
- **Slack** — Slack workspace deployment
- **Web Widget** — Embed in internal portals
- **AI for Work Assistant** — Standalone assistant with scheduling
