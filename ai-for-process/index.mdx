## Platform Capabilities

AI for Process enables organizations to automate complex business processes with AI:

### [Workflows](/ai-for-process/workflows)
Design and deploy AI-powered workflows with a visual builder. Combine AI nodes, integrations, and human-in-the-loop steps for end-to-end process automation.

### [Workflow Nodes](/ai-for-process/nodes)
Use pre-built AI nodes for text generation, image processing, document intelligence, and more. Connect nodes to build sophisticated automation pipelines.

### [Inbox](/ai-for-process/inbox)
Human-in-the-loop request management. Review, approve, and provide input for workflow decisions that require human judgment.

### [Model Hub](/ai-for-process/models)
Access open-source models, deploy fine-tuned models, and connect external model providers. Full control over AI model lifecycle.

### [Prompts](/ai-for-process/prompts)
Design, test, and manage prompts with Prompt Studio. Version control, A/B testing, and collaborative prompt development.

## Key Features

### No-Code Workflow Builder

| Feature | Description |
|---------|-------------|
| **Visual designer** | Drag-and-drop workflow creation |
| **Pre-built nodes** | AI, integration, logic, and utility nodes |
| **Branching** | Conditional paths and loops |
| **Error handling** | Built-in retry and fallback logic |

### AI Capabilities

| Node Type | Capabilities |
|-----------|-------------|
| **Text-to-Text** | Generation, summarization, extraction |
| **Text-to-Image** | Image generation from prompts |
| **Image-to-Text** | OCR, image description, analysis |
| **Audio-to-Text** | Transcription, speech recognition |
| **Doc Intelligence** | Document parsing and extraction |

### Human-in-the-Loop

| Feature | Description |
|---------|-------------|
| **Approval workflows** | Human sign-off on AI decisions |
| **Review queues** | Centralized inbox for pending tasks |
| **Escalation** | Route complex cases to experts |
| **Feedback capture** | Improve AI with human corrections |

### Model Management

| Feature | Description |
|---------|-------------|
| **Open-source models** | Access Hugging Face, custom models |
| **Fine-tuning** | Train models on your data |
| **External providers** | OpenAI, Anthropic, Google |
| **Model versioning** | Track and deploy model versions |

## Use Cases

### Document Processing

Automate document-heavy workflows:

```
[Document Upload] → [OCR Extraction] → [Data Validation] → [Human Review] → [System Update]
```

- Invoice processing
- Contract analysis
- Form extraction
- Compliance review

### Content Generation

Generate and review content at scale:

```
[Brief Input] → [AI Generation] → [Quality Check] → [Human Approval] → [Publish]
```

- Marketing copy
- Product descriptions
- Report generation
- Email drafting

### Data Processing

Transform and enrich data:

```
[Data Input] → [AI Classification] → [Enrichment] → [Validation] → [Output]
```

- Lead scoring
- Sentiment analysis
- Entity extraction
- Data categorization
